(PESONA) Penata Usahaan Stok Opname

Sistem PESONA merupakan sistem yang digunakan untuk mengelola data gudang pada instansi BPMP Provinsi Bali. 
Sistem ini dikembangkan untuk mendigitalisasikan sistem yang biasa digunakan secara manual.

Bahasa pemrograman: PHP
Version: >=8.0.0
Role user: Super Admin, Pimpinan, Admin Barang Habis Pakai (Admin BHP), Admin Aset BMN (Admin BMN), dan Staff
Keunggulan: ringan, praktis, sederhana
